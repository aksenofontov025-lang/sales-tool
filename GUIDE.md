# Помощник продаж — 2К Консалтинг

> Веб-инструмент для менеджеров по продажам (МОП). Формирование коммерческих предложений, квалификация лидов, интеграция с Битрикс24.

**Ссылка:** [https://aksenofontov025-lang.github.io/sales-tool/](https://aksenofontov025-lang.github.io/sales-tool/)

---

## Как пользоваться

### 1. Выберите направление
На главном экране выберите категорию услуг: Боты с ИИ, Битрикс24, AMO CRM, 1С, МойСклад, Сайты, Аналитика или Интеграции.

### 2. Квалифицируйте клиента
Заполните данные клиента (имя, компания, телефон, email). Отметьте чек-лист:
- **Бюджет** — клиент готов платить
- **Сроки** — есть конкретные сроки
- **ЛПР** — общаемся с лицом, принимающим решение
- **Потребность** — есть реальная боль/задача

Статус определяется автоматически: Целевой / Тёплый / Нецелевой.

### 3. Задайте вопросы
Список вопросов для выявления потребностей + красные флаги для распознавания нецелевых обращений.

### 4. Сформируйте КП
- Выберите услуги из прайса (галочками)
- Укажите **свою цену** — кликните на услугу и впишите сумму
- Используйте режим **«Варианты (A/B/C)»** для 2–3 пакетов
- Укажите **скидку** в %

### 5. Настройте сроки
Каждый этап проекта имеет срок в днях. Измените при необходимости — общий срок пересчитается. Сроки отобразятся в PDF как календарный план.

### 6. Отправьте КП
Три способа:
| Способ | Для чего |
|--------|----------|
| 📋 Скопировать текст | Telegram, WhatsApp, мессенджеры |
| 📄 Скачать PDF | Email, формальные предложения |
| 🔗 Прикрепить к сделке | Автоматически в Битрикс24 с PDF |

### 7. Сравните направления
Кнопка **«⚖️ Сравнить два направления»** — когда клиенту подходят два разных решения (например, Бот vs Битрикс24):
- Выберите 2 категории
- Укажите плюсы каждого варианта
- Отметьте рекомендацию
- Скопируйте или скачайте PDF

### 8. История КП
Все сформированные КП сохраняются автоматически в браузере. Кнопка **«📋 История КП»** на главном экране:
- Просмотр прошлых КП
- Быстрое копирование текста
- Удаление ненужных записей

## Советы

- **Своя цена** — всегда можно указать точную сумму вместо диапазона
- **A/B/C пакеты** — клиенты выбирают средний вариант в 70% случаев
- **Скидка** — используйте для мотивации к быстрому решению
- **PDF в Б24** — файл прикрепляется к таймлайну сделки автоматически
- **Офлайн** — инструмент работает без интернета (кроме отправки в Б24)

---

## Технические детали

- Работает в любом браузере (Chrome, Safari, Firefox)
- Оптимизирован для мобильных (iPhone, Android)
- Данные КП хранятся в localStorage браузера (до 50 записей)
- Не требует установки — откройте ссылку и добавьте на рабочий стол
